# Apuntes de etiquetas y atributos utilizados


## <nav>
- Descripción: Representa una sección de navegación con enlaces.


## <a>
- Descripción: Crea enlaces a otras páginas o sitios.
- Atributos :
  - href: indica a dónde lleva el enlace.
  - target="_blank": abre el enlace en otra pestaña.
  - rel="noopener noreferrer": mejora la seguridad en enlaces.


## <header>
- Descripción: Representa la cabecera de la página


## <h1>, <h2>, <h3>
- Descripción: Encabezados de distintos niveles.


## <p>
- Descripción: Define un párrafo de texto.


## <strong>
- Descripción: Resalta texto con importancia (negrita).


## <main>
- Descripción: Contiene el contenido principal de la página.


## <section>
- Descripción: Agrupa contenido relacionado.
- Atributos:
  - class: permite identificar o aplicar estilos.

---

## <img>
- Descripción: Muestra una imagen.
- Atributos:
  - src: ruta de la imagen.
  - alt: texto alternativo.
  - width: ancho de la imagen.
  - class: identificador para estilos.

---

## <footer>
- Descripción: Representa el pie de página.


## <ul>
- Descripción: Lista desordenada.
- Atributos:
  - class: identificador para estilos.

## <li>
- Descripción: Elemento de una lista.


## <button>
- Descripción: Crea un botón interactivo.
- Atributos:
  - onclick: ejecuta una función de JavaScript al hacer clic.


## <div>
- Descripción: Contenedor genérico.
- Atributos:
  - id: identificador único.
  - style: estilos.

---

## <script>
- Descripción: Permite incluir código JavaScript.


## <table>
- Descripción: Crea una tabla.
- Atributos:
  - border: define el borde.


## <tr>
- Descripción: Define una fila de la tabla.


## <th>
- Descripción:encabezado de la tabla.


## <cite>
- Descripción: Se utiliza para citar una fuente o autor.
